# Математический анализ ковариантного сдвига в задачах семантической сегментации

Данный репозиторий содержит программную реализацию к магистерской диссертации по направлению "Математика и компьютерные науки" (ВГУ, 2026 год).

## Цель проекта
Главной задачей разработанного комплекса является не создание модели для устранения ковариантного сдвига, а строгое математическое исследование и демонстрация этого явления на примере деградации масок застройки под действием сезонных изменений (снежного покрова).

## Состав исходного кода
В репозитории представлены ключевые скрипты:
1_kmeans_clustering.py — предварительный спектральный анализ базовой линии методом K-средних.
2_data_pipeline.py — конвейер аугментаций с использованием диэдральной группы симметрий квадрата.
3_train_optimized.py — цикл обучения архитектуры U-Net (ResNet-34) с использованием косинусного отжига и гибридной функции потерь.
4_evaluate_covariate_shift.py — скрипт топологического анализа масок (вычисление площади и дисперсии связных компонент на дискретной решетке).

## Инфраструктура
Обучение производилось в среде Google Colab (GPU NVIDIA T4) с использованием фреймворка PyTorch.
